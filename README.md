# Reto2Niger2.0

Despues de clonar el repositorio de GitHub, realizar npm install en backend (backend) en el frontend (Reto2Niger2.0) y en la base del proyecto (la carpeta principal).

-----------------------------------------------------------------------


Para iniciar el proyecto

Ejecutar este comando en la base del proyecto. 

Este comando inicia todas las partes necesarias del proyecto.

npm run start:all
